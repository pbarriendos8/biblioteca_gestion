Consulta de Libros

En la sección Libros puedes ver un listado de todos los libros disponibles en la biblioteca.

  • Lista de libros disponibles.
  • Utiliza el campo de búsqueda para filtrar por título o autor
  • Como usuario de consulta, las opciones de Nuevo, Editar y Eliminar no están disponibles.

Realizar un Préstamo

Haz clic en Hacer préstamo para registrar un nuevo préstamo.

Se abrirá un formulario donde debes seleccionar el libro y la fecha de entrega.
